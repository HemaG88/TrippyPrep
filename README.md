# TrippyPrep

**TrippyPrep** is an AI-powered placement preparation platform built with Python to help students prepare for campus placements in one place.

### Features
-  10,000+ Aptitude Questions
-  Logical & Visual Reasoning
-  Verbal Ability
-  Coding Practice
-  Company-wise Questions
-  Progress Tracking
-  AI-based Learning Assistance
-  Mock Tests & Analytics
-  Daily Practice Sets
-  Interview Preparation
